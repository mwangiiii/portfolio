Makazi Hub

Makazi Hub is a dynamic rental housing platform designed to connect house listers and hunters seamlessly. The system allows users to list, search, and book rental properties with ease. It features role-based dashboards, advanced search filters by location and category, and a visually appealing user interface built with Bootstrap and Laravel Tailwind components. Additional features include user authentication, property image management, and a booking system that ensures a user-friendly experience for both listers and hunters.
